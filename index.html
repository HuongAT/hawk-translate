What it does

Select. Translate. Done.

Inline bubble

Highlight text on any page and the translation appears in a small popover right next to it. No new tabs, no context switching.

Five providers

Pick what fits: Google (default, no setup), MyMemory, DeepL, Azure Translator, or your own LibreTranslate instance.

Speak it back

Hear the original or translated text. Click the speaker once for normal speed, twice for slow repeat.

Right-click translate

Select text and use the right-click menu to translate without leaving the page.

Toolbar popup

Type or paste anything into the toolbar popup for a quick lookup with full provider control.

Word families

For supported providers, see synonyms and parts of speech alongside the translation.

Translation backends

Use whichever provider you trust.

Google (default) MyMemory DeepL — your API key Azure Translator — your API key LibreTranslate — your endpoint

Your text is sent only to the provider you choose. API keys are stored locally in your browser and never sent anywhere except the matching provider's API.

Privacy

No tracking. No accounts. Nothing collected.

Hawk Translate does not collect, store, or transmit any personal data. There is no analytics, no telemetry, and no remote logging of any kind.

Translation requests go directly from your browser to whichever provider you select. Settings and API keys are stored only on your device.

Read the full privacy policy for details on each provider.
